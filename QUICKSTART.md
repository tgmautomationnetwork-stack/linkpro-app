# 🚀 LINKPRO - QUICK START

## Lokale Entwicklung

### 1. Dependencies installieren
```bash
yarn install
```

### 2. Environment Variables
```bash
cp .env.example .env.local
# Bearbeite .env.local mit deinen API Keys
```

### 3. Development Server starten
```bash
yarn dev
```

App läuft auf: http://localhost:3000

### 4. Build testen
```bash
yarn build
yarn start
```

## Deployment

Siehe **DEPLOYMENT.md** für vollständige Anleitung.

### Quick Vercel Deployment
```bash
# Install Vercel CLI
npm i -g vercel

# Login
vercel login

# Deploy
vercel --prod
```
